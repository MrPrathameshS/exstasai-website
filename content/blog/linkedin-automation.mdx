## Introduction

In this project, we built an AI-powered system that helps professionals
create and schedule LinkedIn posts automatically.

This system is used internally at ExstasAI and is designed to help
users build a personal brand using AI-generated content.

## Features

- AI post generation
- Scheduling
- Content history
- Automation pipeline

## Why we built this

We wanted a system that allows consistent posting without manually
writing content every day.

This led to the creation of our AI Social Automation platform.
